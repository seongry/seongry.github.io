Queue

가장 먼저 넣은 데이터를 가장 먼저 꺼낼 수 있는 구조 -> FIFO 줄을 서는 행위와 유사 Stack이랑은 꺼내는 순서가 반대

알아둘 용어

  • Enqueue: 큐에 데이터를 넣는 기능
  • Dequeue: 큐에 데이터를 빼는 기능

큐가 많이 쓰이는 곳?

멀티태스킹을 위한 프로세스 스케쥴링 방식으 구현하기 위해 많이 사용됨

class Queue {
  _queue;
  constructor() {
    this._queue = [];
  }
  get() {
    return this._queue;
  }
  enqueue(data) {
    this._queue.push(data);
  }
  dequeue() {
    return this._queue.splice(0, 1);
  }
}

const queue = new Queue();

queue.enqueue("typescript");
console.log("Enqueue: ", queue.get());

queue.enqueue("hello");
queue.enqueue("world");
queue.dequeue();
console.log("Dequeue: ", queue.get());
